# AWS Community Day Kenya 2025 Website

This repository contains the source code for the AWS Community Day Kenya 2025 website, a community-organized cloud education event showcasing technical discussions, workshops, and networking opportunities for AWS users in Kenya.

![AWS Community Day Kenya Logo](images/community-day-logo.png)

## Overview

The AWS Community Day Kenya 2025 website serves as the central hub for information about the upcoming event scheduled for June 14, 2025. The site provides details about the event, showcases previous events, and offers opportunities for attendees, speakers, sponsors, and exhibitors to participate.

## Features

- **Event Information**: Details about the upcoming AWS Community Day Kenya 2025
- **Previous Event Showcase**: Interactive display of past AWS Community Day events
- **Venue Information**: Maps and details about the event location
- **Speaker Profiles**: Information about keynote and session speakers
- **Agenda**: Schedule for the day's activities and sessions
- **Sponsor Information**: Details about event sponsors and exhibitors
- **Registration**: Links to register for the event when tickets become available
- **Contact Information**: How to reach the organizing team

## Technology Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- jQuery
- Bootstrap 3.3.7
- Google Maps API
- Font Awesome Icons
- Ionicons
- SlickJS for carousels

## Setup and Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AWS-User-Group-Kenya/community-day-2025.git
   ```

2. Navigate to the project directory:
   ```bash
   cd community-day-2025
   ```

3. Open `index.html` in your browser to view the site locally.
